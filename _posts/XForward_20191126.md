## Paper
- Linux client
  - ` ssh -X user@userip`
  
- Window 10 client
  - use `putty` + `Xming`
  - putty
    - Session: IP address setup
    - SSH -> X11 -> Eanble X11 forwarding (v)
    - SSH -> X11 -> X display location: `localhost:11.0`
  - Xming
    - XLaunch -> Display number: `11` and press next-step until finish
